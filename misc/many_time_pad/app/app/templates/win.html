Remote Access:
If you prefer not to setup a local linux environment, you may SSH into the OSUSEC ctf workserver. Note that this requires an active OSU account. Accessing our environment is easy, but you will not have the freedom to choose your own tools.

  1. Install the OSU VPN and connect.
  2. Open a terminal and generate an SSH key using ssh-keygen -t rsa. A passphrase is not required, you may hit enter to leave it blank.
  3. Copy the public key you just generated using pbcopy < ~/.ssh/id_rsa.pub.
  4. DM the ctf league discord bot the following command $vm_reg username pubkey_copied_from_previous_step, this will generate a user with your intended name, which will authenticate using your keypair.
  5. Wait for approval, or DM an admin asking to have your account quickly approved and registered.
  6. Connect to the workserver with ssh username@cctf-host.osusec.org
